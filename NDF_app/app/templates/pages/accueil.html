Description de l'application


Notes d'une Frondeuse est une application web visant à rendre accessible les articles du même nom écrits quotidiennement dans le journal La Fronde par Caroline Rémy, dite Séverine, entre 1897 et 1899. Cette application est développée dans le cadre du Master 2 Technologies numériques appliquées à l'Histoire de l'École nationale des Chartes.

Les articles proviennent des versions numérisés du journal, disponibles sur Gallica, qui ont été océrisés puis encodés en XML-TEI. RECUPERATION DU XML pour chaque note/pour toutes les notes à des fins d'études textuelles possible. DETAIL DU PROJET SUR LA PAGE A PROPOS.

Fonctionnalités de l'application:

  • Une édition numérique d'un échantillon d'articles choisis, listé sur la page Corpus.
  • Des index de lieux et personnes permettant de naviguer dans les articles en fonction de ceux-ci.
  • Une recherche plein-texte par mot clef défini par l'utilisateur dans l'intégralité des articles.
